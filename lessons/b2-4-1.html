B2 第四章 圓與直線

4-1 圓方程式

紅把圓規腳尖按在方格紙上。圓的每一點，都和中心保持同一段距離。

故事課

固定距離畫出的界線

紅把圓心標在 (2, -1)，圓規張到 3 格，問妳圓方程式怎麼寫。妳寫下 (x + 2)² + (y - 1)² = 3，答錯得很完整，連符號和半徑平方都一起漏掉。

紅沒有碰妳的筆，只把圓規尖端重新按回 (2, -1)。妳看見自己錯了兩件事：中心符號反了，右邊也該是半徑平方。

紅輕打妳手心一下，聲音很輕，卻足夠讓妳把公式重新念出來。「中心是 (h, k)，就是 (x - h)² + (y - k)² = r²。」

  • 處罰：紅輕打手心一次，要求妳把中心與半徑各寫一行，避免把符號和平方一起弄混。紅先確認妳接受這個提醒，妳可以說停。

第一個例子回到圓心 (2, -1)、半徑 3，所以方程式是 (x - 2)² + (y + 1)² = 9。妳把右邊的 9 圈起來，提醒自己它不是半徑，而是半徑平方。

紅接著寫第二個例子：圓心 (-4, 5)，半徑 2。妳慢慢套入公式，得到 (x + 4)² + (y - 5)² = 4。紅問為什麼是 x + 4，妳答：「因為 h = -4，所以 x - h = x + 4。」

第三個例子比較麻煩：x² + y² - 4x + 6y - 12 = 0。紅要妳配方。妳把 x² - 4x 變成 (x - 2)² - 4，把 y² + 6y 變成 (y + 3)² - 9，整理成 (x - 2)² + (y + 3)² = 25。

妳看著圓規轉完一圈，忽然覺得這條界線安靜而堅定。圓上的每個點都離中心一樣遠，沒有哪一個點比較特別，也沒有哪一個點可以偷偷靠近或遠離。這種穩定讓妳想到契約裡那些被寫下來的界線。界線不是為了困住人，而是為了讓人知道哪裡仍然安全。

紅確認剛才的打手心已經結束，才把下一題移近。圓心 (0, 0)，半徑 4。妳寫 x² + y² = 16。紅又問，如果圓心改成 (-3, 2)，半徑仍是 4？妳寫 (x + 3)² + (y - 2)² = 16。這一次妳先說 h = -3、k = 2，再處理符號，沒有讓正負號牽著走。

第三個例子，x² + y² + 8x - 2y + 8 = 0。妳配方：x² + 8x = (x + 4)² - 16，y² - 2y = (y - 1)² - 1，整理得 (x + 4)² + (y - 1)² = 9。中心 (-4, 1)，半徑 3。紅問：「如果右邊整理出負數？」妳答：「那就沒有實際的圓，因為半徑平方不能是負的。」

紅說：「圓方程式先找中心，再看距離。符號不是裝飾，半徑也不是半徑平方。」妳把中心和半徑分成兩行寫，感覺思路也被分清楚了。紅把下一道配方題放在圓規旁：「界線要畫得準，中心先找準。」

今日規則 圓心 (h, k)、半徑 r 的圓方程式為 (x - h)² + (y - k)² = r²；一般式可用配方找中心與半徑。

情境例題

圓心 (1, 4)、半徑 6，方程式為 (x - 1)² + (y - 4)² = 36。
