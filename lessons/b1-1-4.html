B1 第一章 坐標系與函數圖形

1-4 一元二次不等式

紅把拋物線壓回數線。根不是結束，根會把區間切開。

故事課

答案不是只有兩個點

紅寫下 (x - 1)(x - 5) < 0，問解在哪裡。妳很快答：「x = 1 或 5。」答案一出口，妳才意識到自己把整題答錯成上一節的方程式。

紅看著妳，沒有立刻說話。妳的心往下一沉，知道自己把不等式當成等式。紅輕打手心一下：「根只是邊界，不是全部答案。」

  • 處罰：紅輕打手心一次，要求妳把根標在數線上，再測試每個區間的正負。紅先確認妳願意繼續；妳也可以要求停在口頭提醒。

妳把 1 和 5 標在數線上，區間被切成 x < 1、1 < x < 5、x > 5。測試 x = 0，兩因式都是負與負，乘積為正；測試 x = 2，一正一負，乘積為負；測試 x = 6，正與正，乘積為正。所以小於 0 的解是 1 < x < 5。

第二個例子，(x + 2)(x - 3) > 0。根是 -2 和 3，開口向上，拋物線在兩根外側為正，所以 x < -2 或 x > 3。

紅又寫第三題：- (x - 1)(x - 4) > 0。妳先注意前面的負號，開口向下，所以兩根之間為正，答案是 1 < x < 4。紅點頭，這次妳沒有忘記看符號。

妳把 1 和 5 看了很久。剛才答成兩個點，不只是因為粗心，也因為點讓人安心。點很明確，對就是對，錯就是錯；可是區間不是那樣，區間會展開成一整段可能。妳不太喜歡那種感覺，因為一整段可能也代表妳要自己判斷哪裡可以站。

紅看出妳的遲疑，先問：「還能繼續嗎？」妳點頭。紅才把戒尺收回桌邊，指著數線說：「處罰已經結束，現在只剩規則。根把數線切開，妳要做的是測每一段，不是把根當答案交出去。」

紅又把題目換成 (x + 1)(x - 2) ≤ 0。妳先標出 -1 和 2，再測試 x = 0，得到 (1)(-2) < 0。因為題目含等號，所以端點也包含，答案是 -1 ≤ x ≤ 2。紅問：「如果沒有等號呢？」妳答：「端點就不能包含，要寫 -1 < x < 2。」紅點頭，這一次妳不只算出答案，也說出理由。

第三個例子變成 (x + 3)(x - 4) ≥ 0。妳知道開口向上，大於等於 0 在兩側，所以 x ≤ -3 或 x ≥ 4。紅要求妳仍然測試：x = -4 成立，x = 0 不成立，x = 5 成立。測試點像一道保險，讓妳不靠記憶硬撐。

紅說：「不等式問的是一整段範圍。妳不能只抓住它等於零的那一刻。」妳把根畫成分界點，區間像分界點之間的空格，終於知道該選哪一段。紅把互動題放在數線旁：「現在妳自己決定哪一段符合條件。」

今日規則先解等式找邊界，再用數線或開口方向判斷大於、小於的區間。

情境例題

(x - 2)(x - 6) ≥ 0 的解為 x ≤ 2 或 x ≥ 6；因為等號成立，所以端點要包含。
