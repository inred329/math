B1 第一章 坐標系與函數圖形

1-1 數線與絕對值

紅把一條數線貼在桌面。所有後面的圖形，都先從這條直線開始。

故事課

離原點有多遠

紅把 0 標在紙帶中央，左邊是負數，右邊是正數。妳看著 -4，紅問：「|-4| 是什麼意思？」

妳答：「把負號拿掉，所以是 4。」紅的眼神很平靜，卻讓妳知道答案還不夠。紅把鉛筆放在 0 和 -4 之間：「不是變魔術去負號，是問它離原點多遠。」

紅輕打妳手心一下，讓妳把想背口訣的衝動放慢。「先量距離，再寫答案。」

  • 處罰：紅輕打手心一次，提醒妳不要把絕對值只當成去負號。處罰結束後，妳要在數線上標出距離。

妳重新看數線：-4 到 0 有 4 格，所以 |-4| = 4；4 到 0 也有 4 格，所以 |4| = 4。紅說：「絕對值是距離，距離不分左右，所以不會是負的。」

第一個例子，|x| = 3。妳一開始想只寫 x = 3，但紅用指尖按住數線左邊。離 0 有 3 格的位置有兩個：x = 3 和 x = -3。

第二個例子，|x - 2| = 5。紅說這是在問 x 到 2 的距離是 5。妳從 2 往右走 5 格到 7，往左走 5 格到 -3，所以 x = 7 或 -3。

妳看著那條紙帶，忽然覺得它不像一條簡單的線。0 在中間，左邊和右邊都安靜地延伸出去，像紅沒有說出口的那句話：從哪裡開始都可以，但不能假裝自己不在某個位置上。妳曾經很討厭「從頭開始」這四個字，因為它聽起來像承認自己退回原點。可現在紅把原點畫在妳面前，原點不是羞辱，而是一個可以重新量距離的地方。

紅又放到妳面前一張小卡：|x + 3| = 2。妳這次沒有急著答，先把它改成 |x - (-3)| = 2，意思是 x 到 -3 的距離是 2。往右到 -1，往左到 -5，所以答案是 -1 或 -5。紅問：「為什麼有兩個？」妳低聲答：「因為距離只問多遠，不問在左邊還是右邊。」這句話從妳口中說出來時，連妳自己都覺得比剛才穩。紅也補問一句：「剛才那一下還可以嗎？」妳點頭，知道自己也可以說停。

紅把妳的答案旁邊畫上一個小圓點。「這就是第一節要妳服從的步驟：先看中心，再量距離，最後才寫左右兩個位置。答錯可以改，但不能跳過定位。」妳點頭，掌心剛才那一下已經不再佔據注意力，真正留下來的是數線上那兩段一樣長的距離。

紅把下一張題卡放到妳面前，計時器亮起微弱的白光。「這一次，輪到妳自己量。」

紅最後把紙帶收進筆記本。「B1 的第一條規則，是位置。妳要先知道自己站在哪裡，才有辦法往後畫出平面、直線和函數。」

今日規則絕對值表示距離，距離不為負；|x - a| = d 表示 x 到 a 的距離是 d。

情境例題

|x + 1| = 4 等於 |x - (-1)| = 4，所以 x 到 -1 距離為 4，答案是 3 或 -5。
