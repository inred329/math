B1 第一章 坐標系與函數圖形

1-3 二次函數

紅畫出一條拋物線。它會上升、下降，也一定有一個最該被看見的轉折點。

故事課

看見拋物線的轉折

紅寫下 y = (x - 2)² + 3，問妳頂點在哪裡。妳看見最後的 3，立刻說：「(0, 3)。」

紅把方格紙移近。「那是 y 軸截距的影子，不是頂點。」紅輕打妳手心一下，讓妳停下這個太快的判斷。

  • 處罰：紅輕打手心一次，要求妳把二次函數先改寫或讀成頂點式，再回答頂點。

紅指著頂點式 y = a(x - h)² + k。「頂點是 (h, k)。所以 y = (x - 2)² + 3 的頂點是 (2, 3)，不是 (0, 3)。因為平方項最小是 0，當 x = 2 時，y 最小為 3。」

第一個例子，y = (x + 1)² - 4。妳讀成 y = (x - (-1))² - 4，所以頂點是 (-1, -4)，開口向上，最小值是 -4。

紅又寫第二個例子：y = -2(x - 3)² + 5。這次 a 是負的，開口向下，頂點 (3, 5) 是最高點，最大值是 5。妳在圖上畫出向下的拋物線，像一個被倒過來的碗。

第三個例子，y = x² - 4x + 1。紅讓妳配方：x² - 4x + 1 = (x - 2)² - 4 + 1 = (x - 2)² - 3，所以頂點是 (2, -3)。

妳看著 (2, 3)，心裡不太舒服。頂點明明只是圖形上的一個點，卻像把妳最近一直避開的事說了出來：最低點在哪裡，最高點在哪裡，不能用一句「差不多」帶過。妳以前討厭二次函數，是因為它不像直線那樣一路往前；它會轉彎，會先降再升，也會先升再降。那種轉折讓妳想到自己，想到回來重新準備時，每個人都像往前走，只有妳覺得自己繞了一圈。

紅沒有讓妳停在那種情緒裡。紅問：「如果 y = (x - 2)² + 3，x = 0 時 y 是多少？」妳算出 y = 7。紅再問 x = 1、x = 2、x = 3。妳寫出 4、3、4，終於看見頂點不是憑公式硬背，而是兩側對稱時最中間的轉折。紅說：「妳剛才以為 (0,3) 是頂點，是因為只看見最後的 3，沒有看見平方項什麼時候最小。」

紅確認妳能繼續後，把另一條拋物線畫在旁邊：y = - (x + 2)² + 1。妳先讀頂點 (-2, 1)，再說開口向下，所以最大值是 1。紅要妳代入 x = -3 和 x = -1，兩邊都得到 0。妳看著對稱的數字，感覺圖形慢慢從霧裡顯出輪廓。紅提醒妳，若妳需要停，可以先先停下來看表格，不必硬撐著畫完。

最後紅給妳一般式 y = x² - 2x - 8。妳配方成 (x - 1)² - 9，頂點是 (1, -9)。妳原本想說「好低」，但紅先開口：「低點不是羞辱。它只是告訴妳，從哪裡開始往上。」妳把這句話寫在圖旁，忽然覺得自己沒有那麼想把錯題本合上。

紅把下一張拋物線題放到妳面前。「先找轉折，再談答案。這一次，妳自己指出頂點。」

紅說：「二次函數最重要的不是線彎不彎，而是它在哪裡轉身。」妳把「頂點」兩字寫在頁首，像替自己標出一個不再亂跑的中心。紅把下一題放到妳面前：「先找轉身的位置，再提交答案。」

今日規則y = a(x - h)² + k 的頂點是 (h, k)；a > 0 開口向上，a < 0 開口向下。

情境例題

y = x² + 6x + 2 = (x + 3)² - 7，頂點是 (-3, -7)，最小值為 -7。
