B1 第二章 直線方程式

2-3 直線的一般式與點到直線的距離

紅把直線改寫成 Ax + By + C = 0。形式變了，距離也能被量出來。

故事課

距離不能帶著正負情緒

紅寫直線 3x + 4y - 12 = 0，點 P(0, 0)，問點到直線距離。妳代入得到 -12，差點把答案寫成 -12/5。

紅用布帶短暫固定妳的手腕。「距離不會是負的。先寫絕對值，再寫分母。」

  • 處罰：短暫固定手腕，要求妳完整寫出距離公式後才能代數字。固定可隨時停止。

妳寫 d = |Ax1 + By1 + C| / √(A² + B²)。代入 P(0, 0)：d = |3 x 0 + 4 x 0 - 12| / √(3² + 4²) = 12/5。

第一個例子，直線 5x - 12y + 6 = 0，點 (0, 0)。距離是 |6| / √(25 + 144) = 6/13。紅提醒妳，分母是法向量長度，不是隨便除一個係數。

第二個例子，把 y = 2x + 1 改成一般式。妳移項得到 2x - y + 1 = 0。紅說一般式不是更高級，只是把所有項收在同一邊，方便使用距離公式。

第三個例子，點 (1, 2) 到直線 3x + 4y - 6 = 0 的距離為 |3 + 8 - 6| / 5 = 1。妳看著 1，感覺直線離點很近，不再只是抽象符號。

妳看著絕對值，才發現自己剛才差點寫出負距離時，其實心裡也在想負數比較適合自己。好像離目標越遠，就越能證明自己真的落後。紅沒有接妳的情緒，只把公式中的直線畫出來，再從 P 點畫一條垂直線過去。「距離不是評價。」紅說，「它只是量出現在有多遠。」

紅確認固定帶不讓妳不舒服後，才讓妳繼續算第二題：點 (1, -2) 到直線 6x + 8y + 10 = 0。妳先寫完整公式，再代入：|6 - 16 + 10| / √(36 + 64) = 0/10 = 0。紅問：「距離是 0 代表什麼？」妳答：「點在直線上。」這一次，0 不是空白，而是一種精確的位置。

第三個例子，點 (3, 4) 到直線 3x + 4y - 25 = 0。妳算 |9 + 16 - 25| / 5 = 0，也在直線上。紅又把常數改成 -20，距離變成 |9 + 16 - 20| / 5 = 1。只是常數改了，直線的位置就移動了。妳看著兩條平行線，突然明白一般式裡的 A、B、C 不只是符號，它們會決定一條線站在哪裡。

紅鬆開固定帶。「妳可以害怕距離，但不能不量。」妳把公式抄在頁角，旁邊寫下：先一般式，再代點，再絕對值，再除以 √(A²+B²)。紅把下一道距離題放到妳面前：「用公式量，不要靠情緒猜。」

紅說：「距離公式像一把冷靜的尺。它不管妳從哪邊靠近直線，只量最短的垂直距離。」紅把下一題放到妳面前，提醒妳先把直線整理成一般式，再開始量。

今日規則直線一般式為 Ax + By + C = 0；點 (x1, y1) 到直線距離為 |Ax1 + By1 + C| / √(A² + B²)。

情境例題

點 (2, 1) 到 3x + 4y - 20 = 0 的距離為 |6 + 4 - 20|/5 = 2。
