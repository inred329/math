B3 第三章 二元一次不等式及其應用

3-1 二元一次聯立方程組

兩條規則同時存在時，答案不是任意選一條，而是找到它們交會的地方。

故事課

兩條線的交點

紅在方格紙上畫兩條線，一條斜上，一條斜下。紅把筆交給妳：「找出同時滿足這兩條規則的位置。」

題目是 x + y = 7，x - y = 1。妳只看第一條，很快說：「x 可以是 3，y 可以是 4。」紅看著妳：「那第二條呢？」妳代入後發現 3 - 4 = -1，不是 1。

  • 處罰：紅用短暫手腕固定帶讓妳停下來，不准只檢查一條式子。妳可以說停；每寫出一組答案，都要代回兩條方程。

紅說，聯立方程組的答案，必須同時滿足所有方程。妳把兩式相加，得到 2x = 8，所以 x = 4；再代回 x + y = 7，得到 y = 3。這次代入第二條：4 - 3 = 1，成立。

紅又寫 2x + y = 11，x + y = 7。妳用消去法，兩式相減得 x = 4，再得 y = 3。紅說：「消去法就是讓其中一個未知數暫時安靜，先看另一個。」

第三個例子是 y = 2x + 1，x + y = 10。妳用代入法，把 y 換成 2x + 1，得到 3x + 1 = 10，x = 3，y = 7。兩條線在 (3,7) 交會。

妳終於懂了，聯立不是多一條麻煩，而是多一條能縮小可能的規則。當限制一起出現，答案反而更清楚。這和 B3 的訓練很像：紅不是只給妳更多限制，而是讓妳看見哪些選擇真的同時成立。

紅確認手腕固定帶只是提醒，沒有讓妳想停下，才寫第三組：x + y = 4，2x + 2y = 8。妳很快發現第二條只是第一條的兩倍，所以有無限多組解。紅說：「兩條線重合時，不是一個交點，而是一整條線。」

接著紅寫 x + y = 4，x + y = 6。妳停住，說：「同樣左邊不可能同時等於 4 和 6，所以無解。」紅讓妳畫圖，兩條平行線永遠不相交。妳終於知道聯立方程不只會有一組答案，也可能無限多，或完全沒有。

紅又補了一題：2x - y = 1，x + y = 8。妳把兩式相加，得到 3x = 9，所以 x = 3，再代回得 y = 5。紅要妳代回兩條式子，兩邊都成立，妳才把答案圈起來。這個圈不再像倉促的結論，而像通過兩道規則後留下的位置。

妳把三種情況寫下：相交一點、重合無限多、平行無解。紅把下一題放到妳面前：「每個答案都要代回所有規則。只滿足一條，不算通過。」

今日規則聯立方程組的解要同時滿足每一條方程；常用代入法與消去法。

情境例題

x + y = 9，x - y = 3，相加得 2x = 12，所以 x = 6，y = 3。
