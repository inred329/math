B3 第三章 二元一次不等式及其應用

3-3 線性規劃

很多限制交在一起時，答案藏在可行區裡。紅要妳學會在限制內找最好的選擇。

故事課

限制交出的可行區

紅把兩種複習任務寫在紙上：整理筆記 x 份，練習題 y 組。每份筆記要 1 小時，每組題要 2 小時，今晚最多 8 小時；而且至少要完成 1 份筆記和 1 組題。

妳立刻說：「那我全做題，效率最高。」紅指著「至少 1 份筆記」那行：「妳又把不喜歡的限制消掉了。」

  • 處罰：紅短暫固定妳的手腕，要求妳先把所有限制列完，才能開始比較最大值或最小值。妳可以說停，固定只用來阻止妳跳步驟。

限制是 x ≥ 1、y ≥ 1、x + 2y ≤ 8。紅要妳畫出可行區。妳先畫三條邊界，再找交點：(1,1)、(6,1)、(1,3.5)。如果只考慮整數，可能點包含 (1,1)、(2,1)、(4,2) 等等。

紅寫目標函數：得分 P = 3x + 5y。紅說：「線性規劃通常是在可行區的頂點找最大或最小。」妳把頂點代入：P(1,1)=8，P(6,1)=23，P(1,3.5)=20.5。若不限整數，最大在 (6,1)。

第二個例子，若限制為 x + y ≤ 10、x ≥ 0、y ≥ 0，目標 P = 2x + 4y，頂點是 (0,0)、(10,0)、(0,10)，最大值在 (0,10)，P = 40。

妳看著方格紙，忽然明白紅為什麼說限制不是敵人。沒有可行區，妳根本不知道哪些選擇能做；沒有目標函數，妳也不知道要怎麼比較。

紅鬆開固定帶。「成熟不是沒有規則，而是在規則裡仍然能選。」

妳看著可行區的頂點，突然明白第三部為什麼一路都在談限制。限制不是叫妳閉上眼睛接受，而是先把不能做的地方排除，留下可以真的選擇的區域。沒有 x ≥ 0、y ≥ 0，題目會失去現實；沒有 x + 2y ≤ 8，妳會以為什麼都能無限拿。

紅又給一個變形：限制 x + y ≤ 10、x ≤ 6、y ≤ 8、x ≥ 0、y ≥ 0，目標 P = 4x + 3y。妳先找頂點 (0,0)、(6,0)、(6,4)、(2,8)、(0,8)，再代入 P：0、24、36、32、24，所以最大值在 (6,4)，P = 36。紅問：「為什麼只看頂點？」妳答：「線性目標在多邊形可行區的最大或最小通常出現在頂點。」

第三個例子，若目標改成 P = x + 5y，妳重新代入同一組頂點，最大變成 (2,8)，P = 42。可行區沒變，目標改了，最佳解也改了。妳突然覺得這很像自己的複習：限制一樣，但目標不同時，選擇也會不同。

紅把下一張規劃題放到妳面前：「先列限制，後找可行區，最後比較目標。不要在還沒知道能不能做以前，就急著問哪個最好。」計時器重新啟動，妳先寫下所有限制，才讓筆尖碰到目標函數。

今日規則線性規劃先列限制、畫可行區，再把目標函數代入頂點比較最大或最小。

情境例題

限制 x + y ≤ 6、x ≥ 0、y ≥ 0，目標 P = x + 2y，頂點 (0,0)、(6,0)、(0,6) 中最大值在 (0,6)，P = 12。
