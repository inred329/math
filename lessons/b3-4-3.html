B3 第四章 指數與對數

4-3 對數

對數是在問：底數要乘幾次，才會到達眼前這個數？

故事課

反過來問的指數

紅寫下 2^3 = 8，又在旁邊寫 log_2 8。妳盯著那個 log，像看見一個陌生的門牌。

「log_2 8 是多少？」紅問。妳猜：「4？」紅搖頭：「不要猜。把它翻成指數句。」妳答錯得很小聲，因為那個 log 讓妳一瞬間忘了它其實是在問指數。

  • 處罰：紅輕打手心一次，要求妳每看到對數都先翻譯成「底數的幾次方等於真數」。紅確認妳願意繼續後才執行，妳也可以說停。

妳寫：log_2 8 = x，意思是 2^x = 8，所以 x = 3。紅說：「對數不是新的怪物，它是指數的反問。」

第二個例子：log_3 81 = x，就是 3^x = 81，所以 x = 4。第三個例子：log_10 1000 = 3，因為 10^3 = 1000。

紅補上條件：底數 a 要大於 0 且不等於 1，真數要大於 0。妳問為什麼真數不能是負數。紅把指數圖形指給妳看：「a^x 的值永遠大於 0，所以反過來問時，真數也必須大於 0。」

妳把定義寫下：若 a^x = N，則 log_a N = x。忽然發現對數不是要把妳推進更深的迷霧，而是讓妳能反向追問：這個結果，是幾次乘法走到的？

紅確認剛才那一下只是提醒後，讓妳自己翻譯第四題：log_4 1 = x。妳一開始想跳過，因為真數 1 看起來太小，不像答案。紅只指著定義。妳寫 4^x = 1，所以 x = 0。紅說：「任何合格底數的 0 次方都是 1。」

第五題是 log_5 (1/25)。妳先翻成 5^x = 1/25，知道 1/25 = 5^-2，所以 x = -2。原來對數可以是負的，只要真數仍然大於 0。妳在旁邊寫下：真數不能負，但對數值可以負。這句話把妳剛才的混亂切開了。

紅又問：「log_1 8 可以嗎？」妳想起底數不能等於 1，搖頭。紅說，因為 1 的任何次方都是 1，無法反問出唯一的指數。妳重新整理條件：底數大於 0、不等於 1；真數大於 0。紅再補 log_2 16 和 log_4 16 的比較，前者是 4，後者是 2，底數不同，問出的次方也不同。紅把下一題放到妳面前：「先翻譯，再回答。對數只是反方向的指數。」

今日規則log_a N = x 等價於 a^x = N；a > 0、a ≠ 1、N > 0。

情境例題

log_2 32 = 5，因為 2^5 = 32。
