B2 第一章 三角函數

1-4 正弦、餘弦函數的圖形

紅把圓上的高度一格格搬到座標平面。波形不是裝飾，而是角度走過的痕跡。

故事課

圓被拉成一條波

紅把一條細線沿著單位圓繞過去，再把線拉直貼在方格紙的 x 軸上。妳看著圓上的點被投到圖上，像把轉動拆成一段段時間。

「y = sin x 和 y = cos x 一樣嗎？」紅問。

妳看見兩條都像波，便回答：「應該差不多，只是名字不同。」紅安靜地把筆放下，那種安靜比直接說錯更讓妳心口一緊。

紅讓妳把左手掌平放在紙邊，戒尺輕落一下。「波形長得像，不代表起點一樣。」紅說，「先寫關鍵點。」紅先確認妳接受這個提醒；若妳說停，就改成只重畫表格。

  • 處罰：紅輕打手心一次，提醒妳不要用外觀取代關鍵點。處罰結束後，妳要親手補完 0°、90°、180°、270°、360° 的函數值。

妳先寫正弦：sin 0° = 0，sin 90° = 1，sin 180° = 0，sin 270° = -1，sin 360° = 0。紅把這些點連起來，波從 0 開始，上升到 1，再回到 0，下降到 -1，最後回到 0。

接著是餘弦：cos 0° = 1，cos 90° = 0，cos 180° = -1，cos 270° = 0，cos 360° = 1。妳發現它不是跟正弦重疊，而是從最高點開始。

紅說：「兩者振幅都是 1，週期都是 360°。差別在起點，也就是相位。」紅把 sin x 的圖往左推 90°，它就和 cos x 重合。妳終於明白，差一個名字背後其實差了一段角度。

第二個例子，紅問 sin 450°。妳先把 450° 減 360° 得 90°，所以 sin 450° = 1。第三個例子，cos 540° 等於 cos 180°，答案是 -1。週期讓遠處的角度被帶回熟悉的位置。

妳把兩條波線描深。它們不像單純的曲線，更像紅在提醒妳：同樣的起伏，可以因為起點不同，變成完全不同的故事。妳以前以為只要結果相似，過程就可以省略；可是 sin 和 cos 明明同樣週期、同樣振幅，卻因為起點差了 90°，整條曲線都錯開了。

紅看著妳停筆，先問：「剛才那一下還可以嗎？」妳點頭。紅才繼續：「處罰不是為了讓妳記得疼，是讓妳回到關鍵點。現在把圖形說完整。」妳照著表格念：sin 從 0 開始，90° 到 1；cos 從 1 開始，90° 到 0。兩者都每 360° 重複一次。

紅又把圖形變成 y = 2 sin x。妳先說振幅變成 2，關鍵值從 0、1、0、-1、0 變成 0、2、0、-2、0。接著紅寫 y = -cos x，妳知道它會把 cos 圖形上下翻轉，所以 x = 0 時是 -1，x = 180° 時是 1。妳第一次覺得圖形不只是背五個點，而是能被規則推動。

第三個例子，紅問 cos 900°。妳把 900° 減去兩圈 720°，得到 180°，所以 cos 900° = -1。妳把「週期」兩字寫在箭頭旁，像把很遠的角度拉回手邊。

紅把下一題放在圖形下方。「看起點，看週期，看有沒有被放大或翻轉。計時器重新開始。」

今日規則 sin x 與 cos x 的振幅都是 1、週期都是 360°；sin 從 0 開始，cos 從 1 開始。

情境例題

因為 720° 是兩個完整週期，所以 sin 720° = sin 0° = 0，cos 720° = cos 0° = 1。
