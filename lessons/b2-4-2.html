B2 第四章 圓與直線

4-2 圓與直線的關係

紅把直尺放到圓旁。直線與圓的距離，決定它們相交、相切，或彼此錯過。

故事課

不要用眼睛猜擦過還是穿過

紅畫一個圓心在原點、半徑 5 的圓，再畫直線 x = 5。妳看著圖，覺得直線像碰到圓，又像只差一點，便說：「大概有兩個交點？」

紅把直尺壓在圖上，目光沒有離開妳的答案。「大概不是數學。」紅讓妳把手掌貼在桌面，戒尺輕點一下。「量距離。」

  • 處罰：紅輕打手心一次，要求妳每題先算圓心到直線的距離，再判斷關係。

圓心 (0, 0) 到直線 x = 5 的距離是 5，剛好等於半徑，所以直線與圓相切，只有一個交點。妳把「d = r」寫在旁邊，答案終於穩下來。

第二個例子，還是同一個圓，直線 x = 3。圓心到直線距離 d = 3，小於半徑 5，所以直線穿過圓，有兩個交點。紅把直尺往圓內推，妳看見它真的割出一條弦。

第三個例子，直線 x = 6。距離 d = 6，大於半徑 5，所以沒有交點。紅說：「相交、相切、相離，不是靠視覺猜，是靠 d 和 r 的比較。」

紅又寫一般直線 3x + 4y - 10 = 0。若圓心是 (0, 0)，距離 d = | -10 | / √(3² + 4²) = 2。若圓半徑是 5，因為 2 < 5，所以相交。妳發現距離公式像把斜直線也拉回可以比較的尺度。

紅收起直尺，說 B2 從角度開始，到距離結束。妳學會看方向、看位置，也學會判斷一條線和一個圓到底有沒有碰上。可是當紅把下一題放到妳面前，妳沒有立刻拿筆。妳看著剛才那一下落在掌心的位置，忽然說：「我想停一下。」

紅的手停在直尺旁。這不是妳第一次答錯，卻是妳第一次在訓練中清楚地把界線說出來。妳的聲音有點緊：「剛才可以，但我不想每次答太快都用打手心。我想先有一次口頭提醒。」說完妳才發現自己握著筆太用力，指節都白了。

紅沒有反駁，也沒有把妳的話當成任性。紅把戒尺放回抽屜，讓桌面空出來。「收到。」紅說，「規則需要共同修正。從下一節開始，第一次跳步驟先口頭提醒；如果妳確認繼續，才進入處罰。妳也可以隨時要求降級。」

妳沒有立刻放鬆。剛才說出口的界線像一條剛畫好的切線，只碰到圓的一點，卻足夠清楚。紅把原題留在桌上，問：「現在還要完成它嗎？」妳看了看圓，又看了看直線 x = 5，慢慢點頭。

妳重新算：d = 5，r = 5，所以相切。再看 x = 3，d = 3 < 5，所以相交。x = 6，d = 6 > 5，所以相離。紅沒有給新的處罰，只讓妳把三個判斷寫完整。這一次，妳覺得自己不是被規則推著走，而是和紅一起把規則放回正確的位置。

紅把下一道題放到妳面前，但沒有按下計時器。「妳來決定什麼時候開始。」妳深吸一口氣，按下按鈕。下一冊 B3 會把這些判斷變成更明確的限制，而妳第一次知道，限制也可以由妳說出口。

今日規則 圓心到直線距離 d 與半徑 r 比較：d < r 相交，d = r 相切，d > r 相離。

情境例題

圓 x² + y² = 16 與直線 x = 4 相切；與 x = 2 相交；與 x = 5 相離。
