B1 第三章 式的運算

3-2 除法原理與餘式定理

紅把多項式切成幾段。能分出去的是商，最後留下的是餘式。

故事課

剩下的不是商

紅寫 f(x) = x² + 3x + 2，問除以 x + 1 的餘式。妳把式子因式分解成 (x + 1)(x + 2)，然後答：「x + 2。」

紅用布帶短暫固定妳的手腕，讓妳看著自己的答案。「x + 2 是商，不是餘式。餘式是分完之後剩下的部分。」

  • 處罰：短暫固定手腕，要求妳先寫除法原理，再分辨商與餘式。固定可隨時停止。

紅寫：f(x) = (除式)(商式) + 餘式。這題 f(x) = (x + 1)(x + 2) + 0，所以餘式是 0。妳把 0 圈起來，才發現能整除時餘式不見了。

第一個例子，x² + 5x + 7 除以 x + 2。紅教妳用餘式定理：除以 x - a 的餘式是 f(a)。x + 2 = x - (-2)，所以算 f(-2) = 4 - 10 + 7 = 1，餘式是 1。

第二個例子，f(x) = 2x² - 3x + 4，除以 x - 3 的餘式是 f(3) = 18 - 9 + 4 = 13。妳不用完整長除，也能知道最後剩下什麼。

紅又提醒，若除式不是一次式，餘式次數必須比分母低。妳在筆記上寫下：餘式不是隨便剩，是被規則限制過的剩。

妳看著「餘式」兩個字，心裡有一點刺。剩下來的東西總讓妳不安，像自己從國外回來後也被留在原本的進度外。可是紅沒有讓妳把這種感覺攤開太久，只把除法原理重寫一次。商式不是被剩下的，餘式也不是被丟掉的；它們只是各自有位置。

紅確認妳可以繼續後，給妳第三題：f(x) = x³ - 2x + 5，除以 x - 2。妳不用長除，直接算 f(2) = 8 - 4 + 5 = 9，所以餘式是 9。紅問：「為什麼可以只代 2？」妳答：「因為除以 x - a 時，f(a) 裡的除式那一段會變成 0，剩下的就是餘式。」紅點頭，這一次妳不是背定理，而是說出它為什麼成立。

紅又把除式改成 x + 3。妳立刻把它看成 x - (-3)，代入 f(-3)。若 f(x) = x² + x + 1，餘式是 9 - 3 + 1 = 7。紅說：「符號要看清楚。x + 3 不是代 3，是代 -3。」妳在紙上寫下這句，因為它太容易被急著證明自己懂的妳忽略。

最後紅把手腕固定帶鬆開，讓妳自己把除法原理完整念一遍。妳說：被除式 = 除式 × 商式 + 餘式；餘式次數小於除式；除以 x - a，餘式是 f(a)。紅把下一題放到妳面前：「這一次，分清楚誰被帶走，誰留下。」

紅說：「除法原理其實很像交代責任。哪一段被除式帶走，哪一段留下來，都要清清楚楚。」

今日規則f(x) = 除式 × 商式 + 餘式；除以 x - a 的餘式為 f(a)。

情境例題

f(x) = x² - 4x + 6 除以 x - 2 的餘式是 f(2) = 4 - 8 + 6 = 2。
