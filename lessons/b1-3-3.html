B1 第三章 式的運算

3-3 因式分解與分式

紅把式子拆開給妳看。能約掉的不是某個字，而是完整的因式。

故事課

要約掉整個因式

紅寫下 (x² - 9) / (x - 3)。妳看見上下都有 x，就想把 x 約掉。紅輕打妳手心一下，讓妳的筆停在分式旁。

「不能跨加減亂消。」紅說，「先因式分解，才能看見完整因式。」

  • 處罰：紅輕打手心一次，要求妳先把分子分母因式分解，再決定能否約分。

妳把 x² - 9 分成 (x - 3)(x + 3)，所以原式 = (x - 3)(x + 3)/(x - 3)。在 x ≠ 3 的前提下，可以約掉完整的 (x - 3)，剩下 x + 3。

紅特別圈起 x ≠ 3。「分式的分母不能是 0。即使化簡後看起來沒分母，原本的限制也不能忘。」妳在旁邊寫下限制，像替答案補上一道邊界。

第一個例子，x² + 5x + 6 = (x + 2)(x + 3)。若分式是 (x² + 5x + 6)/(x + 2)，在 x ≠ -2 時可化成 x + 3。

第二個例子，(2x² - 8)/(x + 2)。妳先提出 2：2(x² - 4) = 2(x - 2)(x + 2)，再和分母約掉 x + 2，得到 2(x - 2)，限制是 x ≠ -2。

第三個例子，(x + 1)/(x + 1 + 2)。紅讓妳停下來，因為分母是 x + 3，不含完整的 x + 1 因式，不能約。妳終於懂得，約分不是看長得像，而是看乘法結構。

妳看著 x ≠ 3，心裡忽然一緊。原來有些限制就算化簡後看不見，也仍然存在。這不像錯題本上那些被擦掉的痕跡；它更像妳以為自己能靠「重新開始」抹去的過去。紅沒有讓妳沉進去，只問：「限制為什麼要保留？」妳答：「因為原式裡分母不能是 0。化簡不能改變原本不被允許的值。」

紅確認妳狀態穩定後，才把第二張分式卡放到妳面前：(x² + 2x + 1)/(x + 1)。妳先分解成 (x + 1)²/(x + 1)，在 x ≠ -1 時化成 x + 1。紅又問：「如果題目是 (x² + 2x + 1)/(x + 2) 呢？」妳停下來，沒有再亂約。分子是 (x + 1)²，分母是 x + 2，沒有共同因式，所以不能約。

第三個例子，(x² - x - 6)/(x - 3)。妳分解分子為 (x - 3)(x + 2)，約掉完整的 (x - 3)，得到 x + 2，限制 x ≠ 3。紅讓妳把「完整因式」四個字寫得比其他字更大。妳第一次覺得分式不是故意刁難，而是在要求妳看見結構。

紅把一份只有五題的小測驗放到桌面：數線、座標、直線、二次函數、分式。妳原本想皺眉，卻在第一題下筆前先畫了 0，第二題先標 x 軸，第三題先算斜率。做完時，妳沒有全對，但每一題都能說出自己在哪一步錯。紅看著妳，沒有替妳宣判成功，只問：「現在妳在哪裡？」

妳把筆握緊，慢慢回答：「我好像真的找得到位置。」這句話說出口時，複習室安靜了一下。紅收起代數卡片，又把 B2 的量角器放在桌角。「下一冊會談角度與方向。妳不是重新開始，是帶著工具往前走。」

紅把最後一題移近：「先因式分解，再決定能不能約。完成它，第一部就收進妳自己的筆記裡。」

紅收起代數卡片。「B1 到這裡，妳學會位置、線、圖形和式子。下一冊 B2 會把這些基礎轉成角度與方向。妳不是重新開始，是帶著工具往前走。」計時器重新亮起，最後一題等著妳把第一部整理。

今日規則因式分解先看共同因式、平方差與十字相乘；分式約分只能約完整因式，且原分母限制要保留。

情境例題

(x² - 4x + 4)/(x - 2) = (x - 2)²/(x - 2) = x - 2，但限制為 x ≠ 2。
