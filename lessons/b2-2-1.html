B2 第二章 三角函數的應用

2-1 正弦定理與餘弦定理

紅把直角拿走，只留下斜斜的一個三角形。沒有直角時，妳要換工具。

故事課

不是每個三角形都聽畢氏的話

紅在桌上放三根木條，拼成一個沒有直角的三角形。兩邊分別是 3 和 4，中間夾角是 60°。紅問第三邊的平方是多少。

妳幾乎本能地寫下 3² + 4² = 25。紅看著妳的筆尖，輕聲問：「哪裡有直角？」妳停住，耳根微熱，才發現自己把最熟的工具拿來硬用。

紅讓妳把手放回桌上，戒尺輕落在掌心一下。「工具用錯，比不會更危險。」紅說，「先判斷三角形給了什麼。」

  • 處罰：紅輕打手心一次，要求妳在每題開頭先寫「已知條件」。紅先確認妳同意繼續；這一下只為打斷反射式套公式。

紅寫下餘弦定理：c² = a² + b² - 2ab cos C。這題有兩邊和夾角，所以用它。c² = 3² + 4² - 2 x 3 x 4 x cos 60° = 9 + 16 - 12 = 13。第三邊不是 5，而是 √13。

接著紅換成另一題：角 A = 30°，角 B = 60°，邊 a = 4，求邊 b。這次妳先看見「一組對應邊角」a 和 A 都已知，就知道該用正弦定理：a / sin A = b / sin B。

妳算 b = 4 x sin 60° / sin 30° = 4 x (√3/2) / (1/2) = 4√3。紅點頭：「正弦定理最喜歡成對的邊角；餘弦定理適合兩邊夾角或三邊。」

第三個例子，若 a = 5，b = 7，夾角 C = 120°，妳寫 c² = 25 + 49 - 70 cos 120°。因為 cos 120° = -1/2，所以 c² = 25 + 49 + 35 = 109。非直角的世界也有規矩，只是它不再叫畢氏。

妳把「先看條件」寫在頁首。可是這一次，妳沒有完全順著紅。妳抬頭問：「那如果我一開始就看錯條件呢？是不是又要被罰？」話說出口，複習室裡安靜了一瞬。這是妳第一次不是為答案辯解，而是在問規則本身。

紅沒有立刻回答。紅先把戒尺放回桌面，確認剛才那一下已經結束，也確認妳不是在逞強。然後紅說：「如果妳是看錯，我會讓妳重看；如果妳是跳過，我會阻止妳逃。差別在妳有沒有服從步驟。」這句話讓妳心裡稍微鬆開。原來規則也要分辨原因，不是所有錯都被同一種方式處理。

紅又寫一題：已知 a = 7、b = 8、夾角 C = 60°，求 c²。妳先說：「兩邊和夾角，用餘弦定理。」再算 c² = 49 + 64 - 2 x 7 x 8 x 1/2 = 113 - 56 = 57。紅沒有急著點頭，只問：「為什麼不是正弦定理？」妳答：「因為沒有一組完整的對應邊角。」

第三個例子，紅給 A = 45°、B = 60°、a = 10，求 b。妳說有對應的 A 和 a，所以用正弦定理：b = 10 sin 60° / sin 45°。雖然結果帶根號，但妳知道工具選對了。紅說：「工具選對，計算才有意義。」

紅把下一題放在兩條公式中間。「先說用哪一個定理，再動筆。妳可以問規則，但不能省略判斷。」

紅說，數學裡的服從不是盲目照做，而是服從題目真正給妳的訊息。

今日規則 有對應邊角常用正弦定理；有兩邊與夾角，或三邊關係，常用餘弦定理。

情境例題

若 a = 6、b = 8、夾角 C = 60°，則 c² = 36 + 64 - 96 x 1/2 = 52。
