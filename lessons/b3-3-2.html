B3 第三章 二元一次不等式及其應用

3-2 二元一次不等式

一條線把平面分成兩邊。不等式問的，是哪一邊符合條件。

故事課

答案是一整片區域

紅把透明方格墊放在桌上，畫出直線 x + y = 6。紅問：「x + y ≤ 6 的答案在哪裡？」

妳指著線上的一點：「這裡？」紅搖頭：「妳又把不等式當成等式。」妳答錯得太熟練，像是只要看見一條線，就急著把所有答案縮回線上。

  • 處罰：紅輕打手心一次，要求妳先畫邊界線，再選測試點，不准只指一個點當答案。紅先確認妳願意繼續，妳也可以要求停下重畫圖。

紅說，二元一次不等式的解通常是一片區域。先把不等號換成等號，畫出邊界線；再選一個不在線上的測試點，例如 (0,0)。代入 x + y ≤ 6，得到 0 ≤ 6，成立，所以包含 (0,0) 的那一側就是解區域。

紅又寫 2x - y > 4。妳先畫 2x - y = 4，再代入 (0,0)，得到 0 > 4，不成立，所以要選不包含原點的那一側。紅提醒：「大於或小於，不含等號時，邊界線要畫虛線；含等號時，畫實線。」

第三個例子是 y ≥ x - 2。妳畫出 y = x - 2，測試 (0,0)，得到 0 ≥ -2 成立，所以選原點那一側，並畫實線。這次妳沒有急著圈一個點，而是把整片區域輕輕塗上。

紅說：「限制不是只讓妳站在一個位置。有時它給妳一整片可行的地方，只是妳要先知道哪邊被允許。」

妳看著被陰影標出的半平面，心裡反而安定了一點。限制不是把妳釘死在一個點上，而是告訴妳哪些地方可以試。紅確認妳同意繼續，也確認剛才那一下只是提醒後，讓妳自己補一題：y ≥ x - 1。妳先畫 y = x - 1，因為有等號，所以是實線；再代入 (0,0)，得到 0 ≥ -1，成立，所以陰影在包含原點的一側。

第三個例子，x - 2y < 4。妳先畫 x - 2y = 4，因為沒有等號，所以畫虛線。代入 (0,0)，得到 0 < 4，成立，所以選原點那一側。紅問：「虛線代表什麼？」妳答：「邊界上的點不包含。」這一次，妳沒有把整片區域縮成一個點。

紅再給 y < -x + 3。妳畫出邊界線，因為沒有等號所以用虛線；代入 (0,0)，得到 0 < 3，成立，所以陰影包含原點。妳看著三張不同的陰影圖，終於知道二元不等式不是在找某個孤單答案，而是在找一片被規則允許的地方。

紅把透明方格墊放到妳手邊：「先畫邊界，再測試點，再決定陰影。妳要找的是符合條件的一整片地方。」

今日規則二元一次不等式先畫邊界線，再用測試點判斷解區域；含等號畫實線，不含等號畫虛線。

情境例題

x + y ≤ 5 的邊界是 x + y = 5。代入 (0,0) 成立，所以解區域是包含原點的一側。
