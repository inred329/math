B2 第一章 三角函數

1-2 銳角三角函數

紅把三角板立在桌燈前。銳角不只是一個尖尖的角，它會決定妳該看哪一條邊。

故事課

先站到角的位置

紅在紙上畫了一個直角三角形，角 A 被圈在左下方。對邊標成 3，鄰邊標成 4，斜邊標成 5。妳看見三個數字，心裡鬆了一下，覺得這題應該不會太難。

「sin A 是多少？」紅問。

妳答得很快：「4/5。」話一出口，紅沒有立刻說錯，只把鉛筆尖放到角 A 上，讓妳自己從那個角的位置看出去。妳才發現自己答錯了：4 是貼著角 A 的鄰邊，3 才是角 A 對面的邊。

紅把妳的手掌攤開，戒尺輕點一下。「妳剛剛沒有站到角的位置。」紅說，「每次判斷對邊、鄰邊，都要先問：我是從哪個角看出去？」

  • 處罰：紅輕打手心一次，只作為停止跳步驟的提醒。紅確認妳仍願意繼續後，才要求妳把角 A 的對邊、鄰邊、斜邊重新標出來。

妳重新標：斜邊永遠是直角對面的最長邊，所以是 5；角 A 對面的邊是 3；和角 A 夾在一起、但不是斜邊的邊是 4。紅寫下三個比例：sin = 對邊 / 斜邊，cos = 鄰邊 / 斜邊，tan = 對邊 / 鄰邊。

第一個例子回到 3-4-5 三角形。對角 A 來說，sin A = 3/5，cos A = 4/5，tan A = 3/4。紅把錯誤答案 4/5 圈起來，說它不是沒有用，而是屬於 cos A。

紅又畫第二個直角三角形，角 B 的對邊是 5，鄰邊是 12，斜邊是 13。妳先把手按在角 B 旁邊，逼自己從那裡看：sin B = 5/13，cos B = 12/13，tan B = 5/12。這一次，妳沒有急著把最順眼的數字放進答案。

妳把角 B 的位置描深，突然懂了：同一個三角形，換一個角，對邊和鄰邊就會換人。這個發現讓妳有點不自在，因為它太像妳和紅之間正在發生的事。同一件管教，站在紅的位置看，是讓妳停下跳步驟；站在妳的位置看，有時卻像被看穿。角不同，關係也不同。

紅沒有讓妳飄遠，只把三角板轉了半圈。「現在看角 C。」原本對角 A 的對邊，換到角 C 旁邊時變成鄰邊。妳先說出斜邊仍是 5，再重新判斷：若角 C 的對邊是 4、鄰邊是 3，sin C = 4/5，cos C = 3/5，tan C = 4/3。紅問：「所以妳剛才錯在哪裡？」妳答：「我沒有先站到指定角的位置。」

紅確認妳願意繼續後，又拿出一張梯子靠牆的圖。梯長 10，梯腳離牆 6，牆高那段是 8。若角在地面梯腳處，sin = 8/10，cos = 6/10，tan = 8/6。若角換到牆上，對邊和鄰邊就會互換。妳用筆在兩個角旁各畫一個小眼睛，提醒自己每次都要從角的位置看出去。

紅說：「三角函數不是背三個英文縮寫而已。它是在問，某個角和三條邊之間的關係。妳先站錯位置，後面再努力都只是把錯誤算得更整齊。」妳沒有反駁，因為這句話剛好戳中妳最常做的事。

紅把下一題放到妳面前。「先圈角，再找三邊。等妳說出理由，才提交答案。」

今日規則 銳角三角函數要先指定角；sin = 對邊 / 斜邊，cos = 鄰邊 / 斜邊，tan = 對邊 / 鄰邊。

情境例題

若角 C 的對邊為 8，鄰邊為 15，斜邊為 17，則 sin C = 8/17，cos C = 15/17，tan C = 8/15。
