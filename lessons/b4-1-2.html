B4 第一章 排列組合

1-2 直線排列

順序一旦有意義，位置就不再只是空格。紅要妳一格一格數，不准用直覺跳過。

故事課

先後不同，就不是同一件事

紅把五張任務卡排在桌邊，每張都寫著不同內容：背公式、整理錯題、練計算、畫圖、訂正。紅在紙上畫出三個空格，要妳從五張裡挑三張排成今晚的順序。

「有幾種排法？」紅問。

妳想起上一節，脫口而出：「5 + 4 + 3？」

紅把第一格輕輕圈起來。「這不是選一類。第一格要放一張，第二格也要放一張，第三格也要放一張。每一步都要完成。」

妳的臉有些熱。紅沒有催，只把戒尺放在桌上。「今天的處罰很簡單。跳過步驟，就停下來。」

  • 處罰：輕打手心一次。紅先確認妳接受，才用這一下提醒妳先停住，不要急著報答案；接著要妳一格一格說出還剩幾種選擇。

紅指著第一格：「第一格有 5 種選擇。選走一張後，第二格剩 4 種。第三格剩 3 種。所以是 5 x 4 x 3 = 60。」

妳把三個格子重新寫上 5、4、3。這時才看見，排列不是把數字堆上去，而是每一個位置都在問：現在還剩幾種可能？

紅又問：「四本不同筆記排成一列呢？」

妳答：「4 x 3 x 2 x 1 = 24，也就是 4!。」

第三個例子，八張任務卡選三張排順序。妳寫 8 x 7 x 6 = 336。紅補充：「這可以寫成 P(8,3)。從 n 個不同物中取 r 個排成一列，就是 P(n,r)。」

紅再問：「如果五個人排隊，但 A 固定第一位？」妳停了一下，先把 A 放進第一格，剩下四人排後面，所以是 4! = 24。

紅收起卡片。「排列的重點不是大數字，而是位置。當位置有差，先後就有差。」妳點頭，這次沒有急著把答案交給紅。

妳忽然想到，第四部的「選擇」不只是有幾種，也包含先後。先道歉再解釋，和先解釋再道歉，聽起來就不是同一件事；先讀題再計算，和先抓數字就乘，也會得到不同結果。排列把這種差異變成可計算的規則。

紅確認剛才的打手心只是提醒後，又給一題：「六本不同的參考書排成一列，其中數學B必須排第一，有幾種？」妳先固定數學B，剩下五本排列，所以是 5! = 120。紅再問：「如果數學B和錯題本必須相鄰？」妳把兩本綁成一組，和另外四本共五個單位排列，內部還有 2 種順序，所以是 5! x 2 = 240。

紅把下一題放到妳面前：「只要位置不同會造成結果不同，就先想排列。把位置排清楚，再交答案。」

今日規則 順序不同算不同時，用排列；從 n 個取 r 個排列，為 P(n,r)。

情境例題

5 張不同卡片取 3 張排成一列，共有 5 x 4 x 3 = 60 種。
