B3 第二章 方程式與不等式

2-2 一元二次方程式

二次方程式不是只找一個出口。紅要妳學會先看形狀，再決定用因式分解或公式。

故事課

兩個可能的出口

紅在紙上畫了一道拱形曲線。它從左邊落下，碰到數線，又往右邊升起。紅說：「二次方程式常常不像一次方程，只有一個答案。」

紅寫 x^2 - 5x + 6 = 0。妳看見 x^2 就想套公式，卻把 b^2 - 4ac 忘在旁邊，第一行就答錯方向。紅用戒尺輕點妳的手心。「步驟不是裝飾。」

  • 處罰：紅輕打手心一次，要妳在每題開始前先判斷能否因式分解，再寫判別式或公式。紅確認妳仍願意繼續後才執行。

妳重新看 x^2 - 5x + 6。哪兩個數相乘是 6，相加是 -5？是 -2 和 -3，所以 (x - 2)(x - 3) = 0。紅說：「乘積是 0，表示其中一個因式為 0，所以 x = 2 或 x = 3。」

紅又寫 x^2 + 4x + 4 = 0。妳分解成 (x + 2)^2 = 0，答案 x = -2。紅提醒，這叫重根，兩個出口重在同一個位置。

第三題是 x^2 - 2x - 1 = 0。妳找不到整數因式。紅讓妳寫公式：x = [-b ± √(b^2 - 4ac)] / 2a。a = 1，b = -2，c = -1，判別式是 4 + 4 = 8，所以 x = [2 ± √8] / 2 = 1 ± √2。

妳看著兩個答案，想起自己常常希望只有一條路，好讓選擇變得簡單。紅卻說：「二次方程式要妳承認，符合條件的可能不只一個。妳要做的不是先偏愛哪個，而是讓方程自己說話。」

紅確認妳還願意繼續後，才把另一題放到妳面前：x² - 2x + 5 = 0。妳試著分解，找不到合適的兩數。紅沒有讓妳硬猜，而是要妳算判別式 D = b² - 4ac = 4 - 20 = -16。妳說：「小於 0，所以沒有實根。」紅點頭：「不是每條拱線都會碰到 x 軸。」

第三個例子，2x² - 3x - 2 = 0。妳用公式 x = [3 ± √(9 + 16)] / 4 = [3 ± 5]/4，所以 x = 2 或 x = -1/2。紅問：「為什麼這題不用硬分解？」妳答：「可以分解也可以公式，但公式能穩定處理每一題。」

妳重新整理三種情況：D > 0 有兩個相異實根，D = 0 有重根，D < 0 沒有實根。紅把下一題放在判別式旁：「先判斷出口有幾個，再決定怎麼走。」計時器重新啟動，妳先寫下 D，再碰答案。

今日規則一元二次方程可用因式分解、配方法或公式解；判別式 b^2 - 4ac 可判斷根的情形。

情境例題

x^2 - 7x + 12 = 0 可分解為 (x - 3)(x - 4) = 0，所以 x = 3 或 4。
