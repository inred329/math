B2 第三章 向量

3-3 向量的內積

紅讓兩支箭頭交會。內積不只算數字，也判斷兩個方向彼此靠近或疏遠。

故事課

兩個方向的關係

紅在紙上寫 a = (3, 4)，b = (4, -3)。妳一看見兩個向量，先算長度都是 5，便說：「它們應該很像。」

紅把兩支箭頭畫出來，一支往右上，另一支往右下。它們交成直角。妳愣了一下，才知道長度相同不代表方向相同。

紅用布帶短暫固定妳的手腕，讓妳不能立刻把圖擦掉。「別逃回長度。」紅說，「用內積聽方向在說什麼。」

  • 處罰：短暫固定手腕，要求妳先算內積再下結論。固定可隨時停止，目的只是讓妳慢下來。

妳計算 a · b = 3 x 4 + 4 x (-3) = 12 - 12 = 0。紅說，內積為 0 時，兩個非零向量垂直。這比妳用眼睛猜方向可靠得多。

第二個例子，a = (1, 2)，b = (3, 4)。內積是 1 x 3 + 2 x 4 = 11，正數代表夾角是銳角，兩個方向大致同向。紅把箭頭畫在同一側，妳看見數字和圖形對上了。

紅接著寫公式：a · b = |a||b|cos θ。若內積為正，cos θ 為正，夾角小於 90°；內積為負，夾角大於 90°；內積為 0，夾角等於 90°。

妳忽然明白，內積像一個很冷靜的判斷：它不問妳覺得兩支箭頭像不像，只問它們到底朝哪裡用力。這讓妳有點不安，因為妳最近也常在猜紅的方向。紅嚴格時，妳有時覺得紅在推妳往前，有時又覺得紅把妳壓回原地。妳分不清那是銳角、直角，還是某個讓人想退開的鈍角。

紅沒有追問妳在想什麼，只先確認固定可以解除。妳點頭後，紅把布帶鬆開，讓妳自己拿筆。「再一題。」紅寫 a = (2, -1)，b = (-3, 4)。妳算內積 = -6 - 4 = -10，是負數，所以夾角大於 90°。紅問：「圖上會像什麼？」妳畫出一支往右下、一支往左上，兩者確實有反向的成分。

第三個例子，a = (5, 0)，b = (0, -2)。內積是 0，所以垂直。紅說：「垂直不是沒有關係，而是方向完全獨立。」妳把這句話寫下來，覺得它比公式更難忘。

紅又用長度公式帶回夾角：若 |a| = 3，|b| = 4，夾角 60°，內積 = 3 x 4 x cos 60° = 6。妳終於看見坐標算法和角度公式是同一件事的兩種說法。紅把下一題放到妳面前：「先算內積，再判斷角度。不要只靠感覺猜兩個方向的關係。」

今日規則 a · b = x1x2 + y1y2 = |a||b|cos θ；內積為 0 代表非零向量互相垂直。

情境例題

(2, 5) · (4, -1) = 8 - 5 = 3，內積為正，夾角是銳角。
