B3 第四章 指數與對數

4-4 對數函數及其圖形

對數函數只活在 x 大於 0 的地方。紅要妳看見圖形的邊界，也看見反向思考的自由。

故事課

只在正數那一側

紅把 y = 2^x 和 y = log_2 x 畫在同一張紙上。兩條曲線像隔著 y = x 互相對望。

妳問：「log 的 x 可以是 0 嗎？」紅沒有回答，只讓妳把 log_2 0 翻成指數句。妳寫到一半停住，因為沒有任何 2^y 會等於 0。

  • 處罰：紅短暫固定妳的手腕，要妳先寫定義域 x > 0，再畫圖形。妳可以說停；紅不讓妳跳過邊界，也不讓規則越過界線。

紅說，對數函數 y = log_a x 的定義域是 x > 0，圖形通過 (1,0)，因為 log_a 1 = 0。若 a > 1，圖形遞增；若 0 < a < 1，圖形遞減。

紅讓妳算 y = log_2 x 的幾個點：x = 1 時 y = 0，x = 2 時 y = 1，x = 4 時 y = 2。圖形慢慢上升，靠近 y 軸卻不碰到它。紅說，x = 0 是鉛直漸近線。

第二個例子是 y = log_(1/2) x。妳算 log_(1/2) 1 = 0，log_(1/2) 1/2 = 1，log_(1/2) 2 = -1，圖形是遞減的。第三個例子，妳比較 y = 2^x 和 y = log_2 x，發現它們是互為反函數的關係，點 (2,1) 和 (1,2) 像交換位置。

B3 走到這裡，妳終於明白限制不是一道牆。定義域、邊界、漸近線都在告訴妳：哪裡不能去，哪裡才是可行的路。紅收起筆，卻沒有立刻宣布下課，而是把一張空白契約紙放到妳面前。

「第三部結束前，妳自己寫一條規則。」紅說。妳愣了一下，第一反應是想把紙推回去。規則一直是紅給的，妳只要照做、反抗、再照做。可是現在紅要妳自己開口，像把可行區的邊界交到妳手上。

妳想了很久，寫下第一條：每一節開始前，先列出今天最容易跳過的步驟。第二條：答錯時先說錯因，不立刻說「我不會」。第三條：如果處罰或拘束讓我分心，我要先說停，再改成口頭提醒。妳寫完後，手心有點熱，卻沒有把紙藏起來。

紅讀完，只問：「這些規則是為了逃避，還是為了讓妳完成？」妳抬頭，第一次很確定地回答：「是為了完成。」紅把那張紙放進妳的錯題本第一頁。「那從下一冊開始，妳不只是服從我的規則，也要服從妳自己寫下的規則。」

妳重新看 y = log_2 x。它不能碰到 x = 0，但可以在 x > 0 的地方慢慢往前。限制不是把路拿走，而是把路照亮。紅把下一題放到妳面前，計時器沒有立刻啟動。「妳來開始。」妳按下按鈕，像按下自己訂出的第一條規則。

紅說：「妳學會看限制，就能開始做選擇。下一冊，我們談選擇的數量。」

今日規則y = log_a x 的定義域是 x > 0，通過 (1,0)；a > 1 遞增，0 < a < 1 遞減。

情境例題

y = log_2 x 通過 (1,0)、(2,1)、(4,2)，且定義域為 x > 0。
