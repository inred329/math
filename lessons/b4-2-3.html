B4 第二章 機率

2-3 數學期望值

最高獎金很亮，卻不等於值得下注。紅要妳看的不是最幸運的一次，而是長期平均會把妳帶去哪裡。

故事課

不要只看最好的一次

紅今天沒有拿卡片，而是拿出一只透明盒。盒裡有十張紙籤，其中一張寫著 100，其餘九張寫著 0。旁邊放著一枚 20 元代幣。

「抽一次要付 20 元。」紅問，「有十分之一的機會得到 100 元，其他時候得到 0。妳覺得划算嗎？」

妳看著那張 100，忍不住說：「有機會拿到 100，應該可以吧。」

紅把盒子蓋上，沒有讓妳抽。「妳看的不是這個遊戲，而是妳想像中最好的那一次。」

妳的臉有些熱。那句話不只是說抽籤，也像說妳最近的所有努力：總想靠一次漂亮的答案證明自己回到正軌。可是紅不讓妳躲進漂亮的可能裡。紅把紙和筆放正，聲音沉穩：「期望值，是把每種結果乘上它的機率，再全部加起來。」

  • 處罰：紅替妳扣上單手固定帶，限制妳搶先寫答案的動作。每一步妳都要先說出「結果」與「機率」，紅確認後才能寫下來。

紅在紙上寫：100 x 1/10 + 0 x 9/10 = 10。「這代表長期平均下來，每抽一次平均拿回 10 元。但妳每次要付 20 元，所以期望淨收益是 10 - 20 = -10 元。」

妳看著那個負號，終於明白自己剛剛為什麼錯。最高獎金像窗外忽然亮起的光，但期望值看的不是那一道光，而是妳走很久之後，平均會站在哪裡。

紅又拿出骰子。「第二個例子。擲一顆公平骰子，點數就是妳得到的分數。期望值是多少？」

妳依序寫下：1 x 1/6 + 2 x 1/6 + 3 x 1/6 + 4 x 1/6 + 5 x 1/6 + 6 x 1/6。算完後，妳得到 21/6 = 3.5。

「可是骰子不可能擲出 3.5。」妳低聲說。

紅點頭。「所以期望值不是保證出現的結果。它是長期平均。妳擲一次可能是 1，也可能是 6；擲很多次，平均會靠近 3.5。」

第三個例子，紅在紙上寫兩個選項：甲方案固定得到 30 元；乙方案有一半機會得到 80 元，一半機會得到 0 元。妳先算乙方案期望值：80 x 1/2 + 0 x 1/2 = 40。乙的平均較高，但不代表每一次都安全。

「所以期望值能幫我比較長期平均，但不能替我承擔風險。」妳說。

紅鬆開固定帶，指尖在桌面輕敲一下。「這句話就是今天的答案。數學不是把選擇變簡單，而是讓妳知道自己正在選什麼。」

紅確認固定帶沒有讓妳不舒服後，把下一張期望值表放到妳面前。妳這次先畫兩欄：結果、機率。紅說：「每一步都說清楚，再動筆。若妳需要停，就先停下來看表格，而不是停在恐懼裡。」妳把筆放到第一格，終於沒有只盯著最高獎金。計時器重新啟動，下一題要妳算的不是最好的那一次，而是長期平均。

今日規則 期望值 = 各結果乘上對應機率後加總；它代表長期平均，不代表單次保證。

情境例題

抽獎有 1/10 機會得 100 元、9/10 機會得 0 元，獎金期望值是 100 x 1/10 + 0 x 9/10 = 10。若成本是 20 元，期望淨收益為 -10 元。
