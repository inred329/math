B3 第四章 指數與對數

4-2 指數函數及其圖形

底數大於 1 時，圖形往上衝；底數在 0 和 1 之間時，圖形慢慢下降。

故事課

成長與衰退的曲線

紅把 y = 2^x 畫在坐標紙上。妳看見它穿過 (0,1)，往右越升越快，像一件小事在重複之後變成巨大的壓力。

紅問：「y = (1/2)^x 呢？」妳順口說：「也往上。」紅用筆圈住底數 1/2。「底數小於 1，方向會不同。」妳答錯的原因不是不會算，而是把所有指數圖形都想成同一種上升。

  • 處罰：紅短暫固定妳的手腕，讓妳先列 x = -1、0、1、2 的表格，不准憑印象畫圖。妳可以說停，紅也會立刻解除。

妳列表：2^x 在 x = 0 時是 1，x = 1 時是 2，x = 2 時是 4；(1/2)^x 在 x = 0 時是 1，x = 1 時是 1/2，x = 2 時是 1/4。紅說：「兩者都過 (0,1)，都在 x 軸上方，但一個遞增，一個遞減。」

第二個例子，y = 3^x 比 y = 2^x 成長更快。第三個例子，y = (1/3)^x 比 y = (1/2)^x 下降更快。圖形沒有碰到 x 軸，只是越來越靠近，這條線叫水平漸近線 y = 0。

妳看著曲線，忽然覺得它很誠實。重複的選擇會累積，好的壞的都是。紅說：「圖形只是把這件事畫給妳看。」

妳想到自己剛回來時也是這樣。第一天只差一點，第二天也只差一點，可是把那些「只差一點」連起來，就變成妳不敢看的距離。紅沒有讓妳沉下去，只把表格補得更細：x = -1 時，2^x = 1/2；x = 0 時是 1；x = 1 時是 2。指數函數不是從 0 開始，而是穿過 (0,1)。

紅又寫 y = 3 x 2^x。妳先代 x = 0，得到 3；代 x = 1，得到 6；代 x = 2，得到 12。紅說：「前面的 3 會把每個 y 值放大，但底數 2 決定它仍然遞增。」妳把 y = 2^x 和 y = 3 x 2^x 畫在同一張圖上，看到它們都往上，但高度不同。

第三個例子，y = 5(1/2)^x。妳知道它往右遞減，通過 (0,5)，但仍然不會低於 0。紅問：「為什麼不碰 x 軸？」妳答：「因為正底數的指數值永遠大於 0，只會越來越靠近。」紅又讓妳比較 y = 2^x 和 y = 2^x + 1，後者整條往上移 1，水平漸近線也從 y = 0 變成 y = 1。紅把下一題放在表格旁：「先列表，再判斷。不要讓曲線的速度嚇到妳。」

今日規則y = a^x，a > 1 時遞增；0 < a < 1 時遞減；圖形通過 (0,1)，且 y > 0。

情境例題

y = 2^x 遞增，通過 (0,1)，且 x = 3 時 y = 8。
