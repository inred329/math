B1 第三章 式的運算

3-1 多項式的基本概念與四則運算

紅把代數卡片排在桌上。不是所有看起來相近的項，都能被合在一起。

故事課

同類項才可以站在一起

紅給妳 3x² + 2x - x² + 5。妳很快把它寫成 4x + 5，因為妳把 x² 和 x 也合在一起。

紅輕打妳手心一下。「次方不同，就不是同類項。」紅把 x² 卡片和 x 卡片分到兩排，讓妳重新看清楚。

  • 處罰：紅輕打手心一次，要求妳先把同類項分組，再做加減。紅確認妳願意繼續；若妳喊停，這一步改成口頭提醒。

妳重新整理：3x² - x² = 2x²，2x 不能和 x² 合併，所以答案是 2x² + 2x + 5。紅說，多項式的次數看最高次項，這個式子的次數是 2。

第一個例子，(2x² + 3x - 1) + (x² - x + 4) = 3x² + 2x + 3。妳把同類項排齊，運算就不再混亂。

第二個例子，(x + 2)(x + 5)。妳用分配律：x² + 5x + 2x + 10 = x² + 7x + 10。紅提醒，乘法會讓次方增加，最後仍要合併同類項。

第三個例子，2x(3x² - x + 4) = 6x³ - 2x² + 8x。妳看見每一項都要被乘到，不能只挑第一項。

妳看著那些被分成不同排的卡片，忽然懂了自己為什麼常在代數裡出錯。數字和字母混在一起時，妳會想把它們快點收乾淨，好像桌面整齊了，心裡就會比較安穩。可是紅要妳承認，有些東西不能因為看起來礙眼就硬塞在一起。x² 是 x²，x 是 x，它們不是同類。

紅確認妳還能繼續後，把另一組卡片放到妳面前：(4x² - 3x + 2) - (x² + 5x - 6)。妳先把減號分配進去，得到 4x² - 3x + 2 - x² - 5x + 6，再合併成 3x² - 8x + 8。紅問：「哪一步最容易錯？」妳答：「把括號前面的減號漏掉。」紅點頭，讓妳把那個減號用紅筆圈起來。

第三個例子是 (2x - 1)(x + 3)。妳用分配律：2x² + 6x - x - 3 = 2x² + 5x - 3。紅讓妳把四個乘積都畫成箭頭，確認沒有漏掉任何一項。乘法不像加法只是同類項排隊，它會讓每一項都和另一邊每一項相遇。

妳重新拿起最初那題，這次先分組、再合併、最後檢查次數。紅沒有再提醒，因為妳自己把步驟說出來了。紅把下一張多項式題卡放在桌面中央：「規則已經在妳手上。現在讓它替妳整理。」

紅說：「代數的服從，是服從項的種類。誰能合併、誰不能合併，規則比直覺可靠。」紅把下一題放在卡片旁：「先分類，再計算。不要讓整齊的慾望替妳亂合併。」

今日規則同類項是變數與次方都相同的項；加減先分組，乘法用分配律。

情境例題

(3x + 1)(x - 2) = 3x² - 6x + x - 2 = 3x² - 5x - 2。
