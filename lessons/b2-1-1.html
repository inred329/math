B2 第一章 三角函數

1-1 角度的基本性質

紅把量角器放到妳面前。角度不是只有大小，還有起點、方向與轉了幾圈。

故事課

先看方向，再看大小

紅把一枚指針放在圓心，讓它從水平向右的位置開始轉。妳看見指針停在上方，很快說：「九十度。」

紅沒有否定，只把指針反方向轉到同一條終邊。「那這次呢？」妳也想答九十度，卻看見紅的手停在量角器旁。

「角度不是只看停在哪裡。」紅說，「還要看從哪裡開始、往哪個方向轉。」妳把筆握緊，才發現自己剛剛只看終點，沒有看過程。

  • 處罰：紅輕打手心一次，提醒妳每次讀角度都先說出始邊、終邊與方向。紅確認妳接受後才執行，之後立刻回到圖上。

紅在紙上畫出標準位置：角的始邊放在 x 軸正向，逆時針為正，順時針為負。從 x 軸正向逆時針轉到 y 軸正向，是 90°；若順時針轉到同一位置，可以寫成 -270°。兩者終邊相同，稱為同界角。

妳重新看圖，終於懂了為什麼 90° 和 450° 也能指向同一條終邊。450° 是先轉一圈 360°，再多轉 90°。紅寫下：「同界角相差 360° 的整數倍。」

第二個例子，-30° 和 330° 同界，因為 -30° + 360° = 330°。第三個例子，720° 是轉兩整圈，終邊回到 x 軸正向。紅說：「角度記錄的是轉動，不只是結果。」

妳把「方向」圈起來。B2 的第一步不像答案，而像提醒：很多事不能只看最後停在哪裡，還要看妳是怎麼轉到那裡的。B1 教妳定位，B2 卻像開始問另一件事：同樣站在一個地方，妳是順著誰的規則轉過來的？妳不太想承認，自己已經開始在意紅怎麼看妳的答案。

紅像是看出妳的停頓，把指針放回 x 軸正向。「再來。」紅把它逆時針轉到左方。妳先說始邊是 x 軸正向，方向逆時針，轉到 x 軸負向，所以是 180°。紅又把指針順時針轉到同一條終邊。妳這次沒有急著喊 180°，而是說：「也可以是 -180°。」紅點頭，讓妳把兩個角寫成同界。

第三個例子，紅把指針轉到 60°，再繼續多轉一圈。妳寫 420°。紅問：「它和 60° 是不是同一個角？」妳停了一下，回答：「不是同一個轉法，但終邊相同。」這句話讓妳突然想到自己和紅的關係：有些結果看起來一樣，路徑卻完全不同。紅沒有接這個想法，只把題卡放到妳面前，讓妳回到數學。

紅確認妳狀態穩定後，才讓剛才那一下處罰留在過去。「處罰只是提醒，不是答案。」紅說，「真正要服從的是始邊、方向、終邊。」妳把這三個詞寫在頁首，像替自己訂下一個新的讀題順序。

計時器重新亮起。紅把下一個角度寫在卡片上：「這一次，妳自己判斷它轉了幾圈，又停在哪裡。」

今日規則標準位置由 x 軸正向開始；逆時針為正，順時針為負；同界角相差 360° 的整數倍。

情境例題

30° 的同界角可寫成 30° + 360°k。當 k = 1 時是 390°；k = -1 時是 -330°。
